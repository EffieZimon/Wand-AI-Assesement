## Quick Start
1. Clone the repository and navigate into it:

```bash
git clone <git+https://github.com/EffieZimon/Wand-AI---24-Hour-Assesement.git>
cd <Wand-AI>
```
2. Install dependencies
```bash
npm install 
```
3. Run tests cases
```bash
npm cypress run
```
4. Run Test cases in GUI
```bash
npm cypress open 
```
5. Open Videos
```bash 
cypress/videos
```
6. Open Screenshots
```bash
cypress/screenshots
```
